```
/**
 * 20221003题目：
 * 如果出现下述两种情况（任一条件满足，同时满足就算两次无效访问）表示访问无效，
 * 找出无效访问的userId，并按照userId出现的不合法访问次数倒序排序
 * - 同一用户访问时间间隔小于3s
 * - 同一用户10s连续访问三次
 * 限定时间一个自然日
 * 
 * 时间复杂度o(n^^2)
 * 空间复杂度o(n)
 */
```

### 解答：指针法

- step1:计算 用户-访问时间map
- step2:为每个用户计算不合法访问次数，维护三个指针移动
- step3：将UserAndNum实体按照userId出现的不合法访问次数倒序排序，取UserId并返回
